# Engineering Templates

A collection of engineering templates for system design, hardware development, and engineering workflow.

## Purpose

Provide reusable engineering assets:

- **ADR** — Architecture Decision Records
- **SOP** — Standard Operating Procedures
- **Review** — Design & risk review frameworks
- **Document** — Technical documentation templates
- **Checklist** — Hardware & EMC verification checklists

## Structure

```
Engineering-Templates/
├── ADR/
│   └── ADR_Template.md
├── SOP/
│   └── SOP_Template.md
├── Review/
│   ├── Design_Review_Template.md
│   └── Risk_Review_Template.md
├── Document/
│   ├── Technical_Document_Template.md
│   └── Test_Report_Template.md
└── Checklist/
    ├── Hardware_Design_Checklist.md
    └── EMC_Design_Checklist.md
```
